# ⚡️ Drum-Kit-Project ⚡️

This webpage contain a drum kit where each button has different sound, you can play your beat and have fun also share with your friends. 😎

## 📃️ Documentation

This website template is simply made by using pure
* [HTML](https://www.w3schools.com/html/html_intro.asp)
* [CSS](https://www.w3schools.com/css/default.asp)
* [JavaScript](https://www.w3schools.com/js/)
